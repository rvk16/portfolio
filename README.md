# Portfolio - Rahul Vishwakarma

Personal portfolio website showcasing my experience as a Senior Software Engineer specializing in distributed systems and big data.

## Live Site

[https://rvk16.github.io/portfolio](https://rvk16.github.io/portfolio)

## Tech Stack

- HTML5
- CSS3 (Custom properties, Flexbox, Grid, Animations)
- Vanilla JavaScript
- GitHub Pages

## Features

- Responsive design for all devices
- Dark theme with modern UI
- Smooth scroll animations
- Interactive navigation
- Typing effect for titles
- Scroll reveal animations

## Sections

- **About** - Introduction and experience overview
- **Skills** - Technical skills across languages, big data, cloud, databases, observability, and security
- **Experience** - Work history at ServiceNow and Amdocs
- **Projects** - Featured projects with GitHub links
- **Achievements** - Awards and recognitions
- **Contact** - Contact information and social links

## Local Development

Simply open `index.html` in a browser or use a local server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve
```
